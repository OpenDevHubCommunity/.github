# 🚀 OpenDevHub Monorepo

Bem-vindo ao repositório principal do OpenDevHub.

O OpenDevHub é uma comunidade Open Source criada para que estudantes e desenvolvedores possam aprender, colaborar e construir projetos reais utilizando tecnologias modernas do mercado.

---

## 🎯 Objetivo

Criar um ambiente de aprendizado prático onde os participantes possam desenvolver habilidades em:

* Front-end
* Back-end
* Inteligência Artificial
* Banco de Dados
* DevOps
* Cibersegurança
* Trabalho em equipe
* Git e GitHub
